Exercise 1 - Name of the birds

There are 10 birds name to be presented on the screen. Press the first and the last letter of each bird in the list from the top to bottom. When the last letter has been pressed, that bird will be deleted and a new bird will be revealed for you to press.

Exercise 2 - Questions for the Guru

Click on ADD button will reveal a popup form for you to type some questions in. The questions can be categorized as HTML, CSS or Javascript. When clicking the SAVE button, data will be saved into local storage and the table will be updated into the table. You can filter the table using select box.
